# @veldica/publishready-cli

Deterministic writing control CLI.

Part of the [PublishReady](https://veldica.com/publish-ready) ecosystem.

## Installation

```bash
npm install @veldica/publishready-cli
```

## Usage

```bash
npx @veldica/publishready-cli analyze sample.txt
npx @veldica/publishready-cli analyze sample.txt --json
```
