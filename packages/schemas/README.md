# @veldica/publishready-schemas

Unified Zod schemas and inferred types for the PublishReady ecosystem.

Part of the [PublishReady](https://veldica.com/publish-ready) ecosystem.

## Installation

```bash
npm install @veldica/publishready-schemas
```

## Purpose

This package provides the shared Zod schemas and inferred TypeScript types used by the core engine, CLI, MCP server, tests, and public tool contracts.
