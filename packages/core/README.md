# @veldica/publishready-core

Deterministic writing analysis orchestration engine.

Part of the [PublishReady](https://veldica.com/publish-ready) ecosystem.

## Installation

```bash
npm install @veldica/publishready-core
```

## Purpose

This package contains the local analysis pipeline used by the CLI and MCP server: structural metrics, readability formulas, target interpretation, revision levers, profile comparison, and deterministic AI-sounding prose signals.
